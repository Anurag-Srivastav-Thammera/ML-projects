# Machine Learning (ML) Projects 🚀

This repository contains various Machine Learning projects and implementations.

## 📁 Projectss

### 🌸 Iris Flower Classification
- **File:** `proj.ipynb`
- **Description:** A simple yet effective classification project using the Iris dataset. It implements the K-Nearest Neighbors (KNN) algorithm to classify flowers into three species based on their sepal and petal measurements.
- **Algorithm:** K-Nearest Neighbors (KNN)
- **Key Features:**
  - Data loading and exploration using `sklearn`.
  - Feature scaling with `StandardScaler`.
  - Model training with `k=3`.
  - Performance evaluation using Confusion Matrix and Classification Report (achieving 100% accuracy on the test set!).

## 🛠️ Requirements

To run the notebooks in this repository, you'll need:
- Python 3.x
- Jupyter Notebook / Lab
- NumPy
- Pandas
- Scikit-learn

You can install the dependencies using:
```bash
pip install numpy pandas scikit-learn
```

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Anurag-Srivastav-Thammera/ML-projects.git
   ```
2. Navigate to the project directory.
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook proj.ipynb
   ```
